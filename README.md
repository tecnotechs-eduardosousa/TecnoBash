# TecnoBash

TecnoBash é um pequeno sistema com a finalidade de acelerar processos de terminal e git do dia a dia para os desenvolvedores da TecnoTech Sistemas.

### Disclaimer:
Tendo em vista que o TecnoBash é um sistema direcionado para processos internos da TecnoTech Sistemas, este software é de direito desta empresa, não de quaisquer funcionários que tenham criado e/ou contribuído para tal projeto.
